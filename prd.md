# AI 바이브코딩 무료 비밀특강 랜딩페이지 PRD

## 1. 프로젝트 개요

### 1.1 프로젝트 정보
- **프로젝트명**: AI 바이브코딩 무료 비밀특강 랜딩페이지
- **타겟**: 1인 지식창업자, 소상공인, 비개발자 사업가
- **목적**: 이메일 리드 수집 및 무료 특강 신청 전환
- **예상 런칭일**: 2025년 11월 중순

### 1.2 비즈니스 목표
- **주요 목표**: 무료 특강 신청을 통한 고품질 리드 확보
- **핵심 지표(KPI)**:
  - 페이지 방문자 대비 신청 전환율(CVR) 15% 이상
  - 이메일 수집률 월 300+ 리드
  - 특강 실제 참석률 40% 이상
  - 광고 ROAS 300% 이상

### 1.3 타겟 페르소나
**페르소나 1: 지식창업 준비자**
- 연령: 30-45세
- 직업: 강사, 코치, 컨설턴트
- 고민: 온라인 강의 판매 플랫폼 필요, 코딩 지식 전무
- 욕구: 빠르게 내 웹사이트 만들기, 자동화 구축

**페르소나 2: 소상공인**
- 연령: 35-55세
- 직업: 오프라인 사업자, 프리랜서
- 고민: 제작 비용 부담, 외주 관리 어려움
- 욕구: 직접 만들 수 있는 실용적 기술 습득

## 3. 기술 스택

### 3.1 프론트엔드
**프레임워크**: Next.js 14 (App Router)
- 이유: SEO 최적화, 서버 사이드 렌더링, 빠른 성능

**스타일링**: Tailwind CSS
- 이유: 빠른 개발 속도, 유틸리티 우선, 반응형 간편

**애니메이션**: 
- Framer Motion (페이지 전환, 스크롤 애니메이션)
- React Intersection Observer (스크롤 트리거)

**폼 관리**: React Hook Form
- 이유: 성능 최적화, 간편한 유효성 검사

**아이콘**: Lucide React 또는 Heroicons

### 3.2 백엔드
**API**: Next.js API Routes
- `/api/subscribe` - 이메일 수집 엔드포인트
- `/api/send-email` - 웰컴 메일 발송

**데이터베이스**: 
- **옵션 1**: Supabase (PostgreSQL)
- **옵션 2**: Firebase Firestore
- 추천: Supabase (SQL, 실시간 기능, 무료 티어 넉넉)

**이메일 서비스**:
- **옵션 1**: Resend (추천 - 한국 이메일 도달률 우수)
- **옵션 2**: SendGrid
- **옵션 3**: AWS SES

### 3.3 분석 및 트래킹
- **Google Analytics 4**: 페이지뷰, 전환 추적
- **Meta Pixel**: 페이스북 광고 트래킹
- **Google Tag Manager**: 태그 통합 관리
- **Hotjar** (선택): 히트맵, 사용자 행동 분석

### 3.4 배포 및 호스팅
- **호스팅**: Vercel (추천) 또는 Netlify
- **도메인**: [예시] aivibecoding.com
- **SSL**: 자동 HTTPS (Vercel 기본 제공)

```
스크롤 다운 (문제 공감 → 솔루션 → 증거)
  ↓
CTA 클릭 (Section 8)
  ↓
폼 작성 (이름, 이메일)
  ↓
제출 성공 모달
  ↓
웰컴 메일 수신
  ↓
특강 리마인더 (D-3, D-1, 1시간 전)
  ↓
특강 참석 (Zoom)
```

### 4.2 이탈 방지 전략
**Exit-Intent 팝업** (마우스가 화면 밖으로 나갈 때):
- "잠깐만요! 🎁"
- "지금 신청하면 바이브코딩 템플릿 무료 증정"
- 간단한 이메일 입력 폼 (이름 생략)
